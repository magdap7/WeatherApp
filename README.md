# WeatherApp
Program colletcing history of day temperatures

This program allows you to enter the temperature values of the day. Then you can generate temperature statistics: minimum, maximum, control sum and average, and the number of entered values. There are two ways to save temperature values: to the memory for the time of starting the program and to a text file.

Ten program umożliwia wproawadzie wartości temperatur z danego dnia. Następnie można wygenerować statystyki temperatur: wartość minimalną, maksymalną, sumę kontrolną i średnią oraz liczbę wprowadzonych wartości. Możliwości zapisywania wartości temperatur są dwie: do pamięci na czas uruchomienia programu i do pliku tekstowego.
